# TASK 5: Process Management in Linux

## Objective
To understand Linux process management including monitoring processes, terminating processes, managing services, and monitoring system resources.

## Tools Used
- Linux Command Line Interface (CLI)
- systemctl

## Process Management
- Listed processes using ps
- Monitored processes using top
- Created a process using sleep
- Terminated processes using kill and kill -9

## Service Management
- Used systemctl commands
- SSH service was not available in WSL
- Used cron service to demonstrate start, stop, restart, and enable operations

## Resource Monitoring
- Checked memory using free -h
- Checked system load using uptime

## Learning Outcome
- Learned how to manage Linux processes
- Understood PID and process states
- Gained confidence with Linux commands
